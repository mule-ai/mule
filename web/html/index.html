Mule Event Service

Bridge external webhooks to Mule workflows through Supabase

Overview

This service acts as a bridge between external webhooks and Mule workflows:

  1. External services send webhooks to your Supabase instance
  2. Supabase stores and processes these events
  3. This service listens for new events in Supabase
  4. Events are transformed and sent to the Mule API for processing

To get started, configure your Supabase instance and Mule API connection.
